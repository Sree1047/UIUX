# 🎨 UI/UX Design Projects

A collection of **UI/UX design concepts and product interface designs** created to explore modern user experience, visual design, and interaction patterns.

These projects were designed using **Figma and Framer**, focusing on **layout structure, user flow, visual hierarchy, and product experience** rather than frontend development.

---

## 🚀 Projects Included

### 🍎 Apple.com Clone

A UI concept inspired by the Apple website, focusing on **clean layouts, product presentation, and premium design aesthetics**.

### 🚗 Car Website UI

Modern automotive landing page design featuring **hero sections, product highlights, and smooth user navigation**.

### 🤖 ChatBot UI

User-friendly chatbot interface designed for **AI assistants or customer support systems**.

### 💬 Chat App UI

Messaging interface focusing on **conversation flow, usability, and modern chat design patterns**.

### 📦 Delivery App UI

Mobile application UI for delivery services with **order tracking, navigation, and clean mobile-first design**.

### 🛒 E-Commerce UI

Online shopping platform UI with **product browsing, shopping cart experience, and checkout flow**.

### 💪 Fitness App UI

Health and workout tracking interface designed for **fitness dashboards and activity monitoring**.

### 🎬 Movie Ticket Booking UI

Cinema ticket booking interface including **movie listings, seat selection flow, and booking experience**.

### 🩺 Preventive Health & Wellness App

Healthcare-focused UI concept designed for **wellness tracking and preventive health management**.

### 📊 Social Media Dashboard

Analytics dashboard UI displaying **engagement metrics, activity insights, and social media statistics**.

---

## 🛠️ Design Tools Used

* 🎨 **Figma** – UI design, wireframing, and prototyping
* ⚡ **Framer** – Interactive UI prototypes and design experiments
